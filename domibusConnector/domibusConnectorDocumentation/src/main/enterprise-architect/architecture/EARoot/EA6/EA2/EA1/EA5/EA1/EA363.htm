This object contains an attachment for a message. With every message there can be some documents (mostly PDF's) sent along with. Therefore those documents are attached to the message over this type.  Attributes:  attachment: The data itself in byte[]  name: The name of the attachment. Most usefull usage is the file name of the attachment.  mimeType: The type of the attachment. Example: "text/xml", "application/pdf"  description:<br/>
